How to read serial communication of Arduino and Raspi from Dataplicity

1. Open dataplicity.com
2. Login with user: sensyncbasecrbn@gmail.com	password: makanminggu12
3. Go to dataplicity.com/devices/
4. Look at the device named 'raspberrypi' with type 'Raspberry Pi 3 Model B' and the ribbon at the left of the row. If the ribbon is green, the device is connected to Dataplicity. Then click the device name.
5. At the terminal, type these: 
cd ~/serial
sudo python serial_read.py

6. Enjoy.
